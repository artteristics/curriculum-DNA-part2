ผมเริ่มต้นจากการเป็นอาจารย์ผู้สอนในห้องเรียน ก่อนจะก้าวสู่บทบาทที่เกี่ยวข้องกับการออกแบบหลักสูตร พัฒนาผลลัพธ์การเรียนรู้ และกำหนดมาตรฐานทางวิชาการ จนได้เห็นชัดว่าหลักสูตรไม่ใช่เพียงเอกสารหรือแผนการเรียน แต่เป็น “ระบบนิเวศแห่งการเรียนรู้” ที่ต้องขับเคลื่อนด้วยข้อมูล หลักฐาน และการพัฒนาต่อเนื่องอย่างมีทิศทาง
การนำ AUN-QA มาใช้ในการดำเนินงานของหลักสูตร ทำให้แนวคิด “Curriculum DNA” ชัดขึ้นและใช้งานได้จริงยิ่งขึ้น เพราะ AUN-QA เปรียบเสมือนกรอบถอดรหัสที่ช่วยให้เราเห็นความเชื่อมโยงขององค์ประกอบสำคัญทั้งหมด ตั้งแต่การกำหนดผลลัพธ์การเรียนรู้ให้สอดคล้องกับพันธกิจและความต้องการผู้มีส่วนได้ส่วนเสีย การออกแบบรายวิชาและประสบการณ์การเรียนรู้ให้เกิดผลลัพธ์จริง การประเมินที่สะท้อนสมรรถนะ ไปจนถึงการจัดการทรัพยากร บุคลากร และระบบสนับสนุนให้เอื้อต่อคุณภาพ ผลคือ “ดีเอ็นเอของหลักสูตร” ไม่ได้เป็นเพียงแนวคิดเชิงเปรียบเทียบ แต่กลายเป็นโครงสร้างการทำงานที่ตรวจสอบได้และสื่อสารร่วมกันได้ทั้งทีม
เอกสารฉบับนี้จึงตั้งใจทำหน้าที่เป็นแผนที่เริ่มต้นสำหรับผู้ที่ต้องการเข้าใจ แก้ไข และออกแบบหลักสูตรผ่านเลนส์ของ AUN-QA โดยเน้นการทำงานแบบ evidence-based และ PDCA เพื่อให้การประกันคุณภาพไม่ใช่งานที่ทำเฉพาะช่วงประเมิน แต่เป็นวัฒนธรรมการพัฒนาหลักสูตรในชีวิตจริงของทุกภาคส่วน
และเช่นเดียวกับ “ดีเอ็นเอ” ที่ซับซ้อนและเปลี่ยนแปลงได้ หลักสูตรก็ต้องปรับตัวต่อบริบทที่เปลี่ยนเร็ว Curriculum DNA ภายใต้กรอบ AUN-QA จึงเป็นเพียงบทแรกของการสำรวจระบบพันธุกรรมทางการศึกษาที่ยังมีอีกมากให้เรียนรู้—ทั้งการ “กลายพันธุ์” ของหลักสูตรเมื่อเผชิญ future skills การตอบสนองต่อความเปลี่ยนแปลงเชิงเทคโนโลยีและสังคม และการปรับตัวเชิงระบบเพื่อความยั่งยืนของคุณภาพการเรียนรู้ในระยะยาว
